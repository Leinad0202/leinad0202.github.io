# Currículo (GitHub Pages) — Daniel Alexandre Melicio

Este repositório contém uma página estática (HTML/CSS) para publicar o currículo no **GitHub Pages**.

## Como publicar (passo a passo)

1. Crie um repositório no GitHub com um destes nomes:
   - `daniel-melicio.github.io` (recomendado) **ou**
   - qualquer nome (ex.: `resume`) e depois publique via Pages.

2. Faça upload dos arquivos deste pacote para a raiz do repositório:
   - `index.html`
   - `styles.css`

3. Ative o GitHub Pages:
   - Vá em **Settings → Pages**
   - Em **Build and deployment**, selecione:
     - **Source:** `Deploy from a branch`
     - **Branch:** `main` / `root`
   - Salve.

4. Aguarde a publicação.
   - Se o repo for `daniel-melicio.github.io`, a URL normalmente será:
     - `https://daniel-melicio.github.io/`
   - Caso contrário, será algo como:
     - `https://SEU_USUARIO.github.io/NOME_DO_REPO/`

## Personalização rápida
- Edite o texto em `index.html` (seções Summary, Experience, Skills).
- Ajuste cores e espaçamentos em `styles.css`.

## Observação
A página usa a fonte **Inter** via Google Fonts. Se quiser 100% offline, dá para substituir por fontes do sistema.
